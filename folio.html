Gallery

Parallax backrounds effect on header/ FilmRoll Carousel / Materialize / Design


This is the result of a design exercise focused on creating a demo site about a Galery. The Gallery has 3 running exhibitions: Van Gogh, Monet and Holbein.

The site is a demo and has 2 following functional tabs:

Current Exhibitions
There are 3 sections each one starts with a slider with the works of art that are included in the exhibition.
Artists
A small bio section for each one of the relevant Artists.
Details

For this site I used the Materialize: responsive framework taht is based on Google's Material Design.

I also used the FilmRoll: Jquery Carosuel for the exhibition sliders. Added some css to have the images that are not active grayscaled.

The images that I used were found online.

The Raven

Parallax backrounds effect/ Design/ Poe’s poem.


This is the result of a design exercise focused on creating a site that “narrates” Edgar Alan Poe’s “The Raven”.

Details

By using scrolling parallax background effects the background-image changes as Poe’s poem “The Raven” unfolds.

Reaching the to bottom, the background-image fades into one that shows the raven fly in the sky instead of sitting on the fence.
